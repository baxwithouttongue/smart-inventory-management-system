# Smart Inventory Management System

This is the OOP project containg two tasks:

- Task 1: Smart Inventory Management System (Project Report, Python Files, README, USER GUIDE, Video Link)

- Task 2: Study Report on Min-Heap and Knapsack on Smart Inventory Management System (Study Report, Python Files, README, USER GUIDE, Video Link)
